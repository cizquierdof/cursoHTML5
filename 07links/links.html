Enlaces HTML o Hyperlinks

Un hyperlink es un elemento HTML que cuando se hace click sobre él hace saltar al navegador a otro documento. Para hacer un enlace se marca con la etiqueta <a> y con la siguiente sintaxis.

<a href="url">Texto del link<a>

Vemos un ejemplo y luego lo explico:

<a href="http://www.w3schools.com/html/">Tutorial HTML<a>

  • <a> es la propia etiqueta que indica el enlace.
  • href es un atributo que indica la dirección de destino. URL que pude ser una dirección absoluta o relativa.
  • Texto del link es el texto visible en la página y que si no se ha cambiado, por defecto viene en azul y subrayado.

El texto del enlace no tiene porque ser un texto, puede ser una imagen o cualquier otro elemento HTML. El URL puede ser una página externa si se utiliza una dirección completa, o puede ser local si se utilizan URL relativas, o incluso a la propia página si se utliza con el símbolo #.

Cuando se pasa el cursor del ratón sobre un enlace, pasan normalmente dos cosas:

  • La flecha del ratón se convierte en una pequeña mano.
  • El color del elemento de enlace cambia.

Por defecto en los navegadores los enlaces aparecen como texto subrayado de uno de los siguientes colores:

  • azul si no se ha visitado el sitio
  • púrpura si se ha visitado ya el sitio.

Esto se puede cambiar mediante CSS aplicando estilos a los diferentes atributos de <a>

  • a:link es el propio link
  • a:visited es el atributo del enlace visitado
  • a:hover atributo de cuando se pasa por encima el ratón.
  • a:active atributo de enlace activo

Un ejemplo de modificación de estos atributos por defecto podrían ser:

<style>		
	a:link, a:visited {
		color:#49ade9;
		background-color:orange;
		text-decoration:none;
		}
	a:hover {
		color:orange;
		background-color:#49ade9;
		text-decoration:none;
	}
	a:active {
		color:orange;
		background-color:#C0C0C0;
		text-decoration:none;
	}
</style>

En el código anterior como se puede ver, se cambia el estilo en tres circunstancias; cuando se pasa el ratón sobre él hoover, cuando ya lo hemos pulsado alguna vez: visited y cuando está activo, es decir cuando se a pulsado pero todavía no se a soltado: active.

El atributo target

Este atributo indica al navegador donde queremos que se abra el documento referenciado por href, hay cinco posibilidades:

  • _blank abre el documento en una nueva ventana o pestaña según esté definido en las preferencias del navegador.
  • _self lo abre en la misma página desde donde se hizo el click. Es la opción por defecto.
  • _parent abre el documento en un marco asociado.
  • _top lo abre en el cuerpo total de la ventana.
  • framename abre el documento en un marco determinado.

Si la página esta incrustada en un marco se puede utilizar "_top" para romper el marco.

Seguro que habeis visto otra posibilidades, pero esto no se consigue con HTML sino con JavaScript, y eso ya está fuera del ámbito de este pequeño manual.

Utilizar imágenes como enlaces

Es muy corriente utilizar imágenes como enlaces. Esto se puede hacer mediante el siguiente código:

<!DOCTYPE html>
<html lang="es">
	<head>
		<meta charset="utf-8">
		<title>Estilos enlace</title>
	
	</head>
	<body>
		<h1>Una imagen puede ser un enlace</h1>
		<p>
			<a href="minion.html" target="_blank"  >
				<img src="images/emoticon-happy.png" 
				style="width: 100px; height: 100px; border: 0" /></a>
		</p>
		<article>
			Los enlaces de esta página se les ha cambiado los 
			estilos por defecto
		</article>
	</body>
</html>

El atributo id

En las etiquetas se puede añadir un atributo id que las identifica de forma única por lo que se puede utilizar para crear un marcador que posteriormente se puede usar en un enlace. Esto es, supongamos un párrafo al que queremos hacerle una referencia para encontrarlo rápidamente, esto se haría:

<p id="miParrafo">Este es el párrafo al que queremos hacer referencia</p>

En cualquier punto de la página podríamos poner un enlace a este párrafo de la siguiente manera:

<a href="#miParrafo">Ir al párrafo</a>

Como vemos el id se indica mediante el carácter almohadilla #.
