Formatear texto

Un poco de estilo

Ya hemos visto que con CSS se puede dar color al texto, seleccionar el tipo y tamaño de la fuente e incluso se puede seleccionar el estilo del texto. Sin embargo en HTML existen etiquetas específicas para dar formato al texto. Uno se puede preguntar el porqué utilizar etiquetas si se puede hacer con CSS y viceversa, ¿porque no usar un único modo de hacer esto? La respuesta no es tan sencilla, la utilización de etiquetas para cambiar el aspecto del texto viene desde los principios de html por lo que vamos a encontrarnos con un montón de páginas que lo utilizan, también está el hecho de que si quieres subrayar, o poner en negrita un par de palabras nada más en todo un párrafo no es lógico meterlas dentro de un elemento para luego aplicar una regla para elegir texto en negrita, por ejemplo dentro de un elemento <span>, cuando en realidad ya tengo una etiqueta que lo hace directamente. Hoy día en muchos casos es preferible utilizar CSS para formatear el texto porque permite hacer muchas más cosas, pero es importante conocer las etiquetas de formato de texto.

Por otro lado determinadas etiquetas de formato de texto añaden valor al texto que rodean y puede ser útil para los buscadores. Así que como regla general se utilizaría una regla CSS para formatear un bloque entero de texto como un párrafo o un encabezado, sin embargo para palabras o grupos de palabras en particular convienen más las etiquetas directas.

Formatos de texto

Las etiquetas especificas para dar formato a los textos son::

Tipos de texto normales:

  • negrita <b></b>
  • strong o importante <strong></strong>
  • italica <i></i>
  • enfatizado <em></em>
  • subrayado <u></u>
  • tachado <s></s>
  • marcado <mark></mark>
  • subescrito <sub></sub>
  • superescrito <sup></sup>
  • eliminado <del></del>
  • insertado <ins></ins>

Tipos de texto especiales para representar código o expresiones matemáticas:

  • teletipo <tt></tt>
  • código <code></code>
  • variable<var></var>
  • muestra <samp></samp>
  • teclado <kbd></kbd>

Tipos de texto para citas, comentarios y definiciones:

  • cita <cite></cite>
  • definición <dfn></dfn>
  • abreviatura <abbr></abbr>
  • encomillado <q></q>
  • Información de contacto
    <address></address>

Como puede verse los efectos de varios de estos elementos dan unos resultados similares.

Por último otra etiqueta importante para el formateo de texto es <pre> esta etiqueta es parecida a la etiqueta de párrafo, pero en este caso se preservan los espacios y los saltos de línea. Veamos esto en un ejemplo:

<!DOCTYPE  HTML>
<html>
<head>
  	<title>pre vs p</title>
	<meta charset="UTF-8">
</head>
<body>

        el espaciado             y los 
        saltos de línea no se conservan
	</p> 
 	<pre>
        Sin embargo en 			un  <pre>
        los espaciados	y los
        saltos de línea si se
        conservan
	</pre>
</body>
</html>
