Encabezados <h1> ... <h6>

Los encabezados sirven para distinguir y realzar la importancia de determinados textos como títulos, subtítulos, etc. Los encabezados tienen una relación jerárquica que empieza con el h1 que es el de mayor importancia hasta el h6 el de menor importancia.

Los encabezados se han usado tradicionalmente para resaltar texto como el nombre de los página, títulos de posts, etc. Es muy fácil caer en la tentación de utilizar estos elementos para resaltara textos, pero hoy en día esto no es recomendable. La nueva versión de HTML indica que los encabezados son elementos para estructurar, no para dar formato. A pesar de que el uso de estas etiquetas modifica la apariencia del texto que contienen haciéndolo más grande y resaltado que el resto de texto, no deberían usarse con esta finalidad ya que los buscadores de Internet utilizan los encabezados para indexar la estructura de las páginas web. Los encabezados h1 deben indicar la parte principal del objeto en donde se sitúen, el h2 la segunda en importancia y así el resto.

Ejemplo:

<article>

<h1>Este es el título principal del artículo</h1>

El elemento <head>

Seguramente te habrás percatado de que siempre, en todas las páginas hay una etiqueta <head> "cabeza", no debemos confundir este elemento con los encabezados, son algo muy distinto. <head> no tiene nada que ver con los encabezados que hemos visto antes. Este elemento sirve para incluir los metadatos de la página y no es visible, de hecho no debe estar dentro del elemento <body> que es la parte visible de la página web. Los elementos más corrientes que aparecen en él son:

  • Elemento <title> que sirve para dar nombre a la página, este no es visible en la página pero aparecerá en la pestaña del navegador.
  • Elemento <meta>, sirve para indicar el juego de caracteres.
  • <meta> también incluye toda la información importante para la página pero que no se tiene que mostrar, como palabras clave, una descripción del sitio, autoría de la página y cosas así.

Es importante incluir <meta> con el juego de caracteres, sobre todo en idiomas como el español que utilizan caracteres especiales como acentos o la ñ. De hecho esta hoja tiene definido <meta charset="utf-8"> dentro del elemento <head>, en caso contrario no se verían ni los acentos ni la ñ en la mayoría de navegadores, digo esto porque Chrome si que activa edl juego de caracteres aunque la página no lo indique. Otros elementos muy comunes dentro de <head> son:

  • elemento <style>, indica los estilos CSS que aplicar internamente
  • <link> indica una hoja de estilos CSS externa que se aplicará.
